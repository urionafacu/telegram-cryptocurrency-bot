# Bot for send me notifications about crypto

## Technologies I have used

 - Deno
 - Telegram API
 - Velociraptor CLI

## Setup

```bash
$ cd ./telegram-cryptocurrency-bot
$ deno install -qAn vr https://deno.land/x/velociraptor@1.4.0/cli.ts
$ vr dev
```
